# Textopia - A Text Editor

## Features 
Light mode<br>
Dark mode<br>
Text uppercase<br>
Text lowercase<br>
Copy text to the clipboard<br>
Clear text<br>
Remove white-spaces<br>
Word count<br>
Character count<br>
Text reading time<br>
Alert<br>
Mobile responsive<br>

## Technologies
React<br>
Next.js 13.4.7 (App Router)<br>
TypeScript<br>
Tailwind<br>
Next-Themes<br>
aSwiper<br>
